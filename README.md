# Influ2 Landing Page

Static landing page experiment for Influ2-style account-based marketing positioning.

This is an older public front-end artifact: HTML, CSS, JavaScript, and images exported from a Webflow-style landing page. It is kept public as a lightweight example of landing-page copy and layout work rather than as an actively maintained application.

## Files

- `index.html`: page markup and metadata.
- `css/`: exported styling.
- `js/`: exported page script.
- `images/`: exported visual assets.

## Status

Archived/public reference. No backend, credentials, private customer data, or deployment secrets are included.
